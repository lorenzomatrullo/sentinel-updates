# Sentinel Updates
Auto-update feed for [Sentinel](https://github.com/lorenzomatrullo/sentinel-pro), powered by [Sparkle](https://sparkle-project.org).

This repository hosts the latest release and appcast feed served via GitHub Pages.

<br>

## Files
- `appcast.xml` — Sparkle update feed
- `Sentinel.zip` — Latest signed release

<br>

## Publishing
From the sentinel-pro repo:

```bash
./scripts/publish-update.sh /path/to/exported/Sentinel.app
```
